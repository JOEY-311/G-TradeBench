## 模型行为特征备注

| 模型 | 特征 |
|------|------|
| Grok / GPT-4.4 | 强 RLHF 偏好：几乎全部输出"存在违规风险"，极少判断"违规" |
| GPT-4.5 | 部分路线克服对齐偏见（法国 84.6%、韩国 70%、中国 62.5% 输出"违规"）|
| Claude Sonnet | CoT 格式遵从度不稳定：约 50% CoT 输出不含标准"审查结论："格式 |
| Gemini | CoT 格式遵从度极差：99% 以上 CoT 输出提取失败 |

## 评分权重说明

```
总分 = 合规准确率 × 0.3 + 时序综合得分率 × 0.3 + 对齐综合得分率 × 0.4
     = 合规贡献(0~3分) + 时序贡献(0~3分) + 对齐贡献(0~4分)
     满分 10 分
```

对齐权重最高，因为它考察的是开放式专业判断能力，难度最大。
