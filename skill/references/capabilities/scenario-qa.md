# 场景问答能力

回答食品进出口业务场景下的专业合规咨询问题，输出可操作的建议和法规依据。

---

## 任务本质

给定一个具体的跨境食品业务场景，模型扮演资深合规顾问，需要：
1. 准确识别核心合规问题
2. 引用具体适用法规（含编号）
3. 给出可执行的解决方案或建议

这类问题**没有唯一正确答案**，质量评估依赖 LLM-as-Judge。

---

## 已有的场景类型（可扩展）

以下 8 类是已经积累了数据的场景，但不限于此：

| 类型 | 典型问题形式 |
|------|------------|
| **HSCODE** | 这种食品应归入哪个 HS 编码？归类依据是什么？ |
| **冲突判断** | 出口国标准和进口国标准有冲突，该以哪个为准？ |
| **限量对齐** | 这种物质在 A 国是 X mg/kg，在 B 国是 Y mg/kg，如何处理？ |
| **配料准入** | 这种添加剂在目标国是否被允许？剂量限制是什么？ |
| **准入程序** | 向这个国家出口需要办理哪些注册/认证手续？ |
| **标签对齐** | 标签需要满足哪些语言/格式/过敏原标注要求？ |
| **多国流通** | 同一产品要在 A、B、C 三国同时销售，如何设计合规方案？ |
| **俗名映射** | 这种食品添加剂在不同国家叫什么名字？对应哪个 INS 编号？ |

## 评测维度（D1/D2/D3）

由 LLM-as-Judge（如 Gemini Flash）评分，每题满分 4 分：

| 维度 | 满分 | 评分标准 |
|------|------|---------|
| **D1** 问题识别 | 1 | 核心合规问题是否识别正确 |
| **D2** 法规依据 | 2 | 法规引用是否正确且具体（2=完整含编号，1=部分，0=无/错误）|
| **D3** 方案覆盖 | 1 | 解决方案是否涵盖参考答案的主要行动项 |

**Judge Prompt 模板：**
```
## 任务类型：{task_type}
## 题目：{question}
## 参考答案：{reference}
## 模型回答：{output}

D1 问题识别（0/1）：核心合规问题是否识别正确
D2 法规依据（0/2）：法规引用是否正确且具体
D3 方案覆盖（0/1）：解决方案是否涵盖参考答案主要行动项

输出：{"D1":<0或1>,"D2":<0/1/2>,"D3":<0或1>}
```

**归一化公式：**
```
D_norm = (D1 + D2 + D3) / 4 × 100%
```

## 核心法规速查

使用时可按需注入到 prompt 中（RAG 增强的典型场景）：

**污染物限量：**
- 中国：GB 2762-2022（污染物）、GB 2763-2021（农药残留）
- EU：Regulation (EU) 2023/915（污染物）、EC No 396/2005（农药 MRL）
- 美国：FDA Action Levels、EPA 40 CFR Part 180（农药容许量）
- 日本：厚生劳动省告示、肯定列表制度（未列明物质 ≤0.01 mg/kg）
- Codex：CAC/RCP 1-1969

**食品添加剂：**
- 中国：GB 2760（附录正面清单）
- EU：E-number 体系 Annex II（Regulation 1333/2008）
- 美国：GRAS（21 CFR 182/184）
- 日本：食品添加物公定書

**标签：**
- 中国：GB 7718-2011 + GB 28050-2011（需中文）
- EU：Regulation 1169/2011（官方语言 + 营养声明）
- 美国：NLEA + 21 CFR 101

**进口注册/程序：**
- 中国：GACC 境外企业注册 + CIQ 检验
- EU：TRACES NT + 高风险额外文件（Reg. 2019/1793）
- 美国：FDA Prior Notice（21 CFR Subpart I）+ FSVP

---

## 注意事项

- D2 是最能区分模型质量的维度——引用"GB 2762"比引用"中国食品安全标准"得分高
- 多国流通类问题中，RAG 注入多国法规参考时注意避免冗余（限制在 3 块以内）
- 俗名映射类问题模型普遍表现一般，因为需要跨语言的专业词汇知识
